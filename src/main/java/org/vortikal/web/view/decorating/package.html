This package contains interfaces and classes for response decoration.

Package Specification

To hook decoration into the work flow, to concrete mechanisms are provided. Either explicitly wrap a view with a {@link WrappingView}, or use the {@link MappingViewResolver} to wrap all views resolved by it.

The actual decoration is done with a {@link ViewWrapper}, {@link DecoratingViewWrapper} is our provided implementation.

The {@link DecoratingViewWrapper} takes an array of {@link Decorator decorators}, allowing them to process the response content.

The main decorator is {@link TemplateDecorator}, which has two features. The first is to allow parsing of the response with a {@link NodeFilter}, optionally substituting matched patterns with custom content (i.e. SSI parsing). The other is resolving a {@link Template} and merging the content into this template.
