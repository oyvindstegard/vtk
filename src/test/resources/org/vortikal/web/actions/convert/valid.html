Network - Wireless connection

Set up

command line [as ROOT]:
  1. iwconfig eth1 essid [network name] (key [encryption key]) (configure unencrypted / WEP-encrypted interface)
  2. ifup eth1 (enable interface and acquire IP address from DHCP server)
Alternative (if problem when accessing interface):
  1. ifup eth1
  2. iwconfig eth1 essid [network name] (key [encryption key])
  3. dhclient eth1 (alternative "dhclient3 eth1")
